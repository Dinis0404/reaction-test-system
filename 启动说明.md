# 启动说明

## 方法一：使用启动脚本（推荐）

双击运行 `start-dev.bat` 文件即可自动启动。

## 方法二：使用命令提示符（CMD）

1. 打开 **命令提示符**（不是 PowerShell）
2. 切换到项目目录：
   ```
   cd C:\Users\mrDin\Desktop\website
   ```
3. 运行：
   ```
   npm run dev
   ```

## 方法三：如果遇到 PowerShell 执行策略问题

### 解决方案 A：使用 CMD
直接使用 **命令提示符（CMD）** 而不是 PowerShell

### 解决方案 B：设置 PowerShell 执行策略
以**管理员身份**打开 PowerShell，运行：
```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```
然后重新打开 PowerShell

### 解决方案 C：使用完整路径
如果 npm 在 `D:\node\`，可以使用：
```powershell
& "D:\node\npm.cmd" run dev
```

## 访问网站

启动成功后，打开浏览器访问：
- **本地地址**: http://localhost:3000
- **网络地址**: http://127.0.0.1:3000

## 停止服务器

在运行服务器的终端窗口中按 `Ctrl + C`

## 注意事项

- 确保 Node.js 已正确安装
- 首次运行需要安装依赖（`npm install`）
- 如果端口 3000 被占用，Next.js 会自动使用下一个可用端口（如 3001）
